## Database Structure

### Key Tables:
- **profiles**: User accounts and roles
- **vehicles**: Fleet vehicles with specifications
- **drivers**: Driver information and stats
- **clients**: Customer contact details
- **routes**: Planned delivery routes
- **deliveries**: Individual delivery tasks
- **notifications**: System alerts
- **analytics_snapshots**: Daily performance data

### Security:
- Row Level Security (RLS) enabled on all tables
- Role-based access control (Admin, Manager, Driver)
- JWT authentication
- Secure password hashing
